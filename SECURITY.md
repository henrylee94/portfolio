# Security Policy

This is a static, front-end-only site — no backend, no database, no user data,
no secrets in the repo. The attack surface is small.

## Reporting a vulnerability

Please open a **[private security advisory](https://github.com/henrylee94/portfolio/security/advisories/new)**
on this repository, or a regular issue for non-sensitive reports. Please do **not**
include real personal data in any report.
